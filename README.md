## 💡 Filosofia de Código

Aqui nós seguimos algumas regras simples:

2. **Console.log() é debugger?** SIM! (até eu lembrar que existe o Debug do VS Code).
3. **Versão 1 é melhor que versão nenhuma!** Deploy primeiro, refatora depois (ou nunca).
